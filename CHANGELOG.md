# Changelog

All notable changes to SignalTwin will be documented in this file.

This project follows a simple human-readable changelog. Versioning is currently pre-1.0 and MVP-oriented.

## [0.1.0] - 2026-05-28

### Added

- Hardware-free SignalTwin Scenario Engine.
- Adapter-ready fixture pipeline for BMS, WiFi CSI, thermal, visual, acoustic, and PZT inputs.
- Rule-based building health risk scoring with JSON, Markdown, and RoomCI-compatible exports.
- Static API and dashboard example contracts.
- English and Japanese README entrypoints.
- OSS baseline documentation: license, contribution guide, security policy, code of conduct, changelog, and GitHub templates.
