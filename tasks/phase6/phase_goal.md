# Phase 6 Goal: Minimal MVP Readiness

## Goal

Prove the minimal MVP is complete, deterministic, and documented enough to continue into optional adapters later.

## Scope

- full regression suite
- all sample scenarios run
- generated outputs inspected
- README usage updated
- phase status board closed

## Non-Goals

- No Edge Kit
- No real sensors
- No dashboard/API
- No ML

## Expected Files

- Modify `README.md`
- Create or modify `tests/test_mvp_regression.py`
- Modify `tasks/status.md`
- Modify `tasks/phase6/phase_status.md`

## Done Criteria

- `pytest` passes.
- Each scenario fixture can run through the CLI.
- README contains install/test/run commands that work.
- `tasks/status.md` marks minimal MVP phases done.
